1. Question 1
What two arguments commonly follow the grep command?

Answer: The string to search for and the file to search through

2. Question 2
In Linux, what does the piping command (|) do?

Answer: It sends the standard output of one command as standard input to another command for further processing.

3. Question 3
A security professional enters cp vulnerabilities.txt /home/analyst/projects into the command line. What do they want the operating system to do?

Answer: Copy the vulnerabilities.txt file into the projects directory

4. Question 4
What command creates a new file called failed_logins.txt?

Answer: touch failed_logins.txt
